## 로그
$\log_{2}{(x)}$ : 2를 몇 번 곱해야지 x가 되냐는 뜻
<br/>
$\log_{2}{(32)} = 5$ : 2를 5번 곱하면 x인 32가 된다.

# 재귀함수 시간복잡도
재귀 함수의 main logic * 함수 호출 횟수
- 재귀 함수 안에서 자신을 k번 호출한다면      
  시간복잡도 : $k^n$
