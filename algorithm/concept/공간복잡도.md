# 공간복잡도
문제를 풀 때 시간복잡도와 달리 공간복잡도는 많이 고려하지는 않는다.     
<br/>
<br/>
문제에서 공간복잡도 활용법
- 문제에 나와있는 최대 공간을 배열에 할당
- 문제에 제한 공간을 메모리에 할당

하지만 메모리 제한을 보고 하지는 않는다.  
<br/>
<br/>
